Card War Game (SwiftUI)

## Features

- Random card generation
- Player vs CPU gameplay
- Score tracking system
- Interactive Deal button
- Custom card graphics and background
- Built with SwiftUI

## Technologies Used

- Swift
- SwiftUI
- Xcode
- State Management (@State)

## Skills Demonstrated

- iOS Development
- SwiftUI Layout Design
- State Management
- User Interface Development
- Random Number Generation
- Mobile App Development

## How to Run

- Open the project in Xcode.
- Select an iOS Simulator or connected device.
- Click Run (⌘ + R).

##Gameplay

- Tap the Deal button.
- The player and CPU receive random cards.
- Cards are displayed on screen.
- Scores are tracked throughout the game.
